# Quick Start Guide

## Prerequisites
- Node.js 14+ (recommended: 16+)
- npm or yarn

## Installation

```bash
npm install
# or
yarn install
```

## Development

```bash
react-scripts start
```

The app runs on http://localhost:3000 (or the port specified).

## Build for Production

```bash
react-scripts build
```

## Environment Setup

No environment variables required for basic setup.
