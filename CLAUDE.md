# argon-dashboard-pro-mui

React version of Argon Dashboard 2 PRO by Creative Tim

**Framework:** React >=16.8.0 + Material UI

## Quick Commands

```bash
npm install          # Install dependencies
react-scripts start  # Start dev server
react-scripts build   # Build for production
react-scripts test       # Run tests
```

## Key Directories

- `src/components/` - Reusable UI components
- `src/views/` or `src/pages/` - Page-level components
- `src/assets/` - Static assets (images, styles)
- `src/layouts/` - Layout wrapper components

## Tech Stack

- **UI Library:** Material UI
- **Styling:** Emotion
- **State:** React Hooks
- **Routing:** React Router
- **Build:** Create React App

## Code Conventions

- Functional components with hooks
- Material UI components for UI
- Consistent import ordering (React, libraries, local)
